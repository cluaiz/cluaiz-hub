# Docs

`doc/` contains the official documentation for Cluaiz Skills and Plugins.

These pages explain how to discover, install, create, and publish skills and
plugins for Cluaiz.

Reading order:

1. `doc/overview.md`: What Cluaiz Skills are, what they can do, and how they fit into the Cluaiz ecosystem.
2. `doc/skill-format.md`: Skill directory structure, SKILL.md frontmatter reference, asset linking, and the agent prompt body.
3. `doc/plugin-format.md`: Plugin directory structure, connector types, and MCP integration.
4. `doc/quickstart.md`: Step-by-step guide to creating your first skill.
5. `doc/cli.md`: CLI reference for `cluaiz skill` commands (search, install, update, remove).
6. `doc/publishing.md`: How to publish skills and plugins to the Cluaiz Registry.
7. `doc/contributing.md`: PR requirements, naming conventions, and review process.
